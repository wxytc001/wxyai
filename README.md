# WxyAI - 产业链分析与行情追踪仪表盘

A股/半导体/光通信/PCB/存储产业链实时分析仪表盘。

## 功能模块

- 📊 **产业链分析** - 板块紧缺度、瓶颈速览、关键事件倒计时
- 📈 **行情追踪** - 市场热度排序、涨跌数据
- 📝 **每日复盘** - 早间要闻、机构观点、市场分析
- 📡 **产业链消息** - 实时新闻聚合

## 内容锁定机制

- 基础市场数据：免费开放
- 深度分析/紧缺度/瓶颈信息：需登录解锁
- 支持手机号登录和微信登录
- 磨砂玻璃模糊效果 + SVG锁图标

## 在线访问

- 云端演示：https://wxyai-r5squ3.surge.sh
- GitHub Pages：部署后可通过 https://wxytc001.github.io/wxyai/ 访问

## 使用方式

直接用浏览器打开 `index.html` 即可使用，无需服务器。

## 技术栈

- 纯前端 HTML/CSS/JavaScript（单文件）
- 暗色主题设计
- 响应式布局
